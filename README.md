## 夢璃の小さなアトリエ

**Ciallo～(∠・ω< )⌒★** <img src="img/1771652418893_upscayl_3x_upscayl-standard-4x.png" alt="夢璃ちゃん" width="200" align="right" style="margin-left: 15px;">

はじめまして、**夢璃（ゆめり)** です（´∀｀）

静かに何かを作る時間が好きで、
アニメを追いかけるのも大好きです。

毎日のほとんどは画面の前。
キーボードをカタカタと叩いたり、
好きな物語の世界に浸ったり。
二つの世界をゆっくり行き来しながら過ごしています（´-`）

効率を追い求めるよりも、過程を大切にしたい。
「どれだけ作れたか」よりも、
「どれだけ本当に好きなものを残せたか」を大事にしています。

まだ成長の途中で、
自分の進む道を探しながら歩いています。
ここで、少しずつ。
やさしくて温かい小さな成果を積み重ねていけたら嬉しいです ( ´ ▽ ` )

*本当の成長とは、過去の自分を否定することではなく、
痛みを受け入れたうえで、それでも優しさを選ぶ勇気を持つこと。*
